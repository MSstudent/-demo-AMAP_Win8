2.高德地图相关介绍：

    高德地图API 是一套应用程序接口。通过API，您可以轻松地在自己的应用中定制强大、快速、轻便的地图功能。
    高德地图API 是免费开放的。
    用户在调用高德地图API 必须加上Key。
